# yadoms
Yadoms is an open source project. 

It aims to publish a simple, powerfull, flexible and multiplatforms domotic solution.

## Forum

http://www.yadoms.com/forum


## Features
* Edit
* Many devices support
* RfxCom
* ZWave
* Automation rule
* View customization
* Widgets
* Charts

## Multiplatforms
* Windows
* Linux
* Mac OSX
* Raspberry PI 1,2&3
* *probably most of unix based systems*
