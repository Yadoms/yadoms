i18n-check is a .NET tool.
It requires .NET framework 4.0 to run.
It was developped with MS Visual Studio 2013 Community Edition (need NuGet).

This tool is designed to check that translations are up-to-date for entire Yadoms or just a sub-module (plugin, widget...).
For more information, type :
 i18n-check --help
 
To check entire Yadoms and its sub-modules, just start i18n-check from your {your_Yadoms_path}\sources directory.
